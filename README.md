# XAI-Sample-Implementation
This repo was created to act as a base for my XAI projects

This project forms as a learning checkpoint fo rmy college 3rd yr Mini project.

## What does this include?
- It includes a sample HTML page which inputs details of a student (marks, attendance, study_hours and internals)
- Clicking on predict score, runs the main.py file and generates an output for the given data
- Output along with the reasoning is shown below the input fields

## Key takeaways
- Learnt deploying python files into websites
- Understood how XAI works
- Implementation of XAI in jupyter-notebook
- Connecting XAI to a simple frontend using FAST API
